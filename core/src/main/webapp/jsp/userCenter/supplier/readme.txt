此文件夹存放1B端（厂家）用户中心页面模板：
1、subNav.jsp是1b用户中心2级导航栏和左侧导航栏，被其他1b端用户中心的页面引入
2、authenticationType.jsp是：我的资料-认证类型页面
3、complaint.jsp是：我的产品-收到的投诉
4、fastPublish.jsp是：我的产品-发布新产品
5、index.jsp是：1b端用户中心首页
6、mobile.jsp是：我的资料-登记手机
7、myAnnounce.jsp是：我的消息-厂家公告
8、myInfo.jsp是：我的资料-修改资料
9、myOperation.jsp是：我的消息-操作日志
10、password.jsp是：我的资料-修改密码
11、productList.jsp是：我的产品-产品列表
12、productLoad.jsp是：我的卖家-产品下载记录
13、promise.jsp是：我的资料-服务承诺
14、selfPublish.jsp是：我的产品-自助发布产品
15、taobaoProduct.jsp是：我的卖家-淘宝产品列表
16、vipLoad.jsp是：我的卖家-会员下载统计

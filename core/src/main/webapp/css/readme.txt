相关内容和规则暂定如下，欢迎多多提改善意见（技术中心前端：孙德贵）

一、此文件夹存放网站的CSS样式文件，文件结构同jsp文件夹一一对应

二、CSS相关规范如下：
1、通用样式若bootstrap里面有相应的类，则引入调用；若没有则按照详细分类加入common文件夹里common.css文件里面
2、公共组件（搜索框、漂浮块等）的样式写入1中指明的common文件夹下的common.css文件里，注释清楚
3、每个页面除去公用样式还应该有页面独有样式，每个页面都需要引入
4、id、class命名，小写单词（或缩写），多个单词用“-”连接。注意语义化命名。
5、当id、class作为js钩子时，必须在名字前加前缀“j-”，比如：“j-content”。样式和js钩子的class、id必须分开，严禁公用。
6、禁止使用CSS表达式
7、冒号后留一个空格，花括号前留一个空格，其他除格式必须外均不留空格
